**English Title**: Outdoor Rain Ambience — Relaxing Rain Sounds for Sleep & Stress Relief

**故事簡介**：無主角、純氛圍。四個固定鏡頭由街道到自然，捕捉大雨落地、水花四濺、雨打樹葉、積水倒影的視覺與聲音層次。整體情緒：沉靜、療愈、微帶憂鬱。每段首尾相同，可無縫循環播放。

**總時長**：40秒（4段 × 10秒）｜**風格**：寫實自然主義 × 冷藍灰色調 × 大氣感特寫

## 第1段｜10秒｜街道大雨

```
【風格】寫實都市街道，冷藍灰色調，10秒，16:9，大雨，低角度廣角鏡
【時長】10秒
【畫幅】16:9
【鏡頭】全程固定鏡頭，無任何推拉搖移，只有畫面內容（雨、行人、車）移動
【首尾相同】第0秒與第10秒畫面完全相同：低角度固定鏡頭拍攝濕滑石板路，大雨從上方傾瀉，密集雨滴打地激起白色水花，積水映出路燈橙色光條，無行人無車，純雨景靜止構圖

0-1秒：低角度固定鏡頭，濕滑石板路，大雨傾瀉，密集雨滴打地，積水映出路燈橙色光條。無行人，無車，純雨景。

1-4秒：幾把雨傘從鏡頭右側緩慢移入——深色布料傘面，雨水沿傘緣滾落成細線，滴在地面形成漣漪。行人腳步沉穩踏過積水，鞋底踩起「噗嗤」水花，水珠濺到鏡頭前景。鏡頭不動。

4-7秒：行人離開畫面。一輛汽車從遠處左側駛近，車燈劃破灰濛濛的雨霧，輪胎輾過積水發出「唰——」的長音，車駛出畫面，尾跡漣漪擴散後慢慢平靜。鏡頭不動。

7-9秒：街道回復無人，只剩大雨持續落下。積水漣漪密集交疊，路燈橙色倒影在水面扭曲搖晃。

9-10秒：畫面回到與第0秒完全相同的構圖——低角度石板路，大雨傾瀉，密集水花，路燈橙色光條倒影，無行人無車。

【聲音】大雨傾盆的密集雨聲（主調，高頻噼啪）+ 雨水打地面聲（厚重、持續）+ 雨打雨傘的噗噗聲 + 行人踩水「噗嗤」聲 + 汽車輾過積水的「唰」聲 + 遠處排水渠的流水聲 + 隱約的城市低鳴（車聲、人聲被雨聲壓蓋）+ 無BGM
```

## 第2段｜10秒｜公園樹下

```
【風格】城市公園樹林邊緣，霧藍綠色調，10秒，16:9，中雨，仰角固定拍樹冠
【時長】10秒
【畫幅】16:9
【鏡頭】全程固定鏡頭，無任何推拉搖移，只有雨水、葉片、樹枝在畫面內自然動態
【首尾相同】第0秒與第10秒畫面完全相同：仰角固定鏡頭望向茂密樹冠，雨絲穿過葉縫斜落，葉片靜止輕顫，天空灰白，無風，無水珠滴落

0-1秒：仰角固定鏡頭，望向茂密樹冠。雨絲穿過葉縫斜向左下飄落，葉片靜止輕顫，天空灰白。無風，無水珠滴落。

1-4秒：一片大葉子積水過多，葉尖突然垂下——一道晶瑩水流從葉尖滴落，在空中短暫拉成一條細線，落到下方另一片葉面，激起小水珠四射。鏡頭固定不動。

4-7秒：水珠落定，葉面恢復靜止。雨絲繼續穿過葉縫飄落，樹皮浸濕變深，雨水沿樹幹溝槽緩慢向下流，苔蘚飽滿發亮。鏡頭固定。

7-9秒：風短暫一陣，樹枝輕搖，積在葉面的雨水被甩落一大片——密集水滴在空中散開，打在下方草地。風停，樹枝回靜。鏡頭固定。

9-10秒：畫面回到與第0秒完全相同的構圖——仰角望樹冠，雨絲穿葉縫斜落，葉片靜止輕顫，天空灰白，無風無水珠。

【聲音】雨打樹葉聲（沙沙沙，葉面共振的柔和感）+ 大水珠從葉尖滴落聲（單顆清脆的「叮」）+ 雨水沿樹幹流動的細微水聲 + 風聲輕掠（短暫）+ 樹枝搖曳甩水的「嘩啦」聲 + 公園遠處雨聲背景 + 無BGM
```

## 第4段｜10秒｜雨後初霽

```
【風格】城市街道，冷藍轉暖光，10秒，16:9，雨勢漸停，廣角固定鏡頭
【時長】10秒
【畫幅】16:9
【鏡頭】全程固定鏡頭，無任何推拉搖移，只有光線變化、雨勢減弱、單車經過為畫面動態
【首尾相同】第0秒與第10秒畫面完全相同：固定廣角鏡頭，城市街道全景，細雨持續，天空灰白微透暖光，地面積水映出建築與路燈倒影，無行人無車，靜謐

0-1秒：固定廣角鏡頭，城市街道全景，細雨持續，天空灰白微透暖光，地面積水映出建築、路燈倒影，無行人無車，靜謐。

1-4秒：遠處天邊出現一條淡金橙色光線，穿透雲層斜射在濕透的街道上，積水倒影即時變成金橘色。光線的倒影在水面上拉長、搖晃。鏡頭固定不動，只有光線在變化。

4-7秒：幾顆大雨珠從電線落下——分散的幾個落點，在倒影水面激起漣漪，金光隨之碎裂又重聚。遠處一輛單車騎過，輪胎輾出一道長長的水跡，漸漸消失在畫面遠端。鏡頭固定。

7-9秒：單車消失，街道回復靜止。細雨維持，天空金橙暖光穩定，薄薄霧氣從地面上升，倒影清晰完整。遠處傳來一聲鳥鳴。鏡頭固定。

9-10秒：畫面回到與第0秒完全相同的構圖——廣角街道，細雨持續，天空灰白微透暖光，積水倒影，無行人無車，靜謐。

【聲音】細雨持續聲（輕柔，比前三段明顯弱）+ 殘餘雨水從屋簷滴落聲（稀疏「叮叮」）+ 電線末端水珠滴落的清脆聲 + 雨後靜謐空氣感（低沉環境音）+ 遠處鳥鳴一聲（若有若無）+ 單車輪胎輾過積水的「嘶——」聲 + 無BGM
```

## 製作備注

**色調一致性**
- 第1–3段維持冷藍灰（色溫偏低，飽和度低）
- 第4段允許色溫轉暖，作為情緒出口

**雨量設定建議**
- 第1段：大雨（暴雨感，雨絲密集可見）
- 第2段：中雨（雨打葉面清晰）
- 第3段：超大雨暴雨（水面翻騰、水花炸射、積水溢出）
- 第4段：由細雨轉停（漸進變化）

**串接順序**
```
第1段 → 第2段 → 第3段 → 第4段
（街道大雨 → 樹下中雨 → 水窪特寫 → 雨後初霽）
```

**聲音設計重點**
- 全段：**無BGM**，讓自然雨聲本身構成音景（soundscape）
- 各段聲音層次分明：環境底層音 → 主題音效 → 細節特寫音

**無縫循環技術要點**
- 每段首尾畫面描述完全相同（第0秒 = 第10秒），明確告知 Seedance 首尾一致
- 使用固定鏡頭消除因鏡頭位移導致首尾接合不自然的風險
- 中段動態（行人、車、水珠滴落、風）在9秒前必須完結，讓最後1秒回到靜止雨景

**固定鏡頭執行要點**
- 每段提示詞均明確寫出「固定鏡頭」、「鏡頭不動」、「無任何推拉搖移」
- 所有動態均來自畫面內容：雨、水花、漣漪、行人、車、光線變化
- 若 Seedance 仍出現輕微鏡頭漂移，可在提示詞加上「static camera, no camera movement」強調

**無參考圖使用說明**
- 本系列無固定角色，不需 `@圖片` 角色一致性
- 若需要特定場景風格，可上傳參考場景圖作為 `@圖片1` 風格參考
