# 清明上班圖 — 場景提示詞

**統一設定：**
人物全身宋代古裝（寬袖儒袍、幞頭冠帽、腰帶配飾），使用現代辦公設備（筆電、機械鍵盤、滑鼠、顯示器、手機）。
畫風為古典工筆畫、宋代院畫、清明上河圖仿古設色。古今道具並存，幽默感來自時代錯位。

以下場景可單獨生成靜幀，亦可作為三段分鏡的首幀參考圖輸入 Seedance。

## 場景 A：早上打卡群像（晨霧入口）

```
古典工筆畫風，宋代宮廷院畫風格，清明上河圖仿古設色（赭石、花青、藤黃、墨綠），宣紙底色質感。

【時代錯位】宋代古裝員工排隊刷手機打卡，閘機顯示「09:01 遲到一分鐘」。

【場景】辦公大樓正門外，清晨薄霧，柔和春光帶輕微逆光。
朱紅木構門廊橫樑、石獅子一對，門框兩側裝有現代感應閘機（金屬閘機、LED 數字顯示屏）。
右側一棵桃花樹，落花偶爾飄落青石板路，春意與焦慮並存。

【人物群像】五名古裝員工縱深排列，由遠到近：
- 人物一（最遠，背影）：低頭滑手機，寬袖儒袍隨步伐搖曳，面無表情
- 人物二（中景）：雙手捧建盞熱茶，邊走邊喝，眼神空洞
- 人物三（近景，打卡中）：舉起手機對準感應器，手機螢幕清晰顯示「09:01　遲到一分鐘」，LED 屏閃紅燈，神情尷尬凝固——眉頭輕蹙，嘴角微抽
- 人物四（近景，等候）：站在人物三身後，低頭刷手機，渾然不知前方正在上演人生悲劇
- 人物五（畫面右側，奔跑入鏡）：小跑步趕入，一手提外賣袋（楷書寫「早餐」），一手夾筆電，頭冠略歪向右，寬袖袍飛揚

【構圖】縱深透視，閘機居畫面中軸，桃花樹右側平衡，人物由遠到近漸清晰。
【光線】清晨薄霧柔光，輕微逆光，建築邊緣帶光暈。
工筆線條細膩，人物表情各有戲（尷尬、麻木、焦急），帶輕幽默感。
9:16 直屏。
```

## 場景 B：高層出場宣布罰款

```
古典工筆畫風，宋代院畫，人物動作誇張帶喜劇感，設色暖黃（清晨室內外光線交界），線條工整細膩。

【時代錯位】宋代高層官員展開竹簡宣讀罰款公告，竹簡上寫「遲到一分扣俸銀一兩」。

【場景】辦公大樓入口，木門剛被推開一刻。
畫面右側：厚重朱紅木門半開，室內透出暖橘光與涼爽外光形成冷暖交界。
青石板地面，石獅子在遠景，桃花瓣飄落地面。

【主角：高層主管】
- 服飾：頭戴高帽紗冠（宋代官帽），身著深靛藍廣袖官袍，腰繫玉帶，衣襬端整，氣場壓人
- 姿態：雙腳穩立，雙手展開一卷竹簡，目光威嚴掃視眾人
- 神情：嚴肅凜然，嘴角帶一絲「早已料到」的淡然，無憤怒，只有從容
- 竹簡特寫文字（楷書工整）：
  「凡遲到一分者，扣俸銀一兩。
   遲到兩分者，扣俸銀兩兩。
   累計三次，罰抄《季度報告》三遍。」

【配角：遲到員工二人，站於左側，表情崩潰】
- 09:01 員工：臉色慘白，單膝微彎，手機滑落差點跌地，喃喃自語（嘴型：「一兩銀……」）
- 09:02 員工（剛衝入者）：外賣袋停在半空，眼神呆滯，頭冠歪斜，整個人如石化

【構圖】高層居右側（主視覺），遲到二人居左側（副視覺），木門居中作為空間分隔，形成「宣判者 vs 受刑者」的古典對仗構圖。
【光線】室外清晨冷光 vs 室內暖橘燈光，在門口形成冷暖分界線，高層站在暖光一側，遲到者站在冷光中。
工筆線條保持院畫端莊，高層威嚴與遲到者崩潰形成喜劇對比。
9:16 直屏。

```

```
姿態：側坐窗邊，雙手輕放膝上，目光落在面前已進入省電模式的筆記型電腦黑色螢幕上。

桌上道具：
- 已黑屏的筆電（省電模式，螢幕如鏡面）
- 滑鼠靜置，手未觸碰
```

## 場景五：三人加班群像（深夜）

```
古典工筆畫風，宋代人物群像，夜景，中近景。
```
