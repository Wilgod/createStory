# 清明上班圖 — 打工古詩集

以古詩詞形式，記錄現代打工人的一日。
可用作影片字幕、旁白、或畫卡文字。

---

### 詩一：早上打卡

**五言絕句 · 〈朝行〉**

```
晨鐘催起身，
霧中赴職去。
打卡差一秒，
魂落九幽處。
```

**白話意譯：** 晨鐘響起不得不起床，霧茫茫中趕去上班。打卡機差一秒就遲到，嚇得魂都快飛了。

**建議用途：** 第一段開場字幕（0-3秒），畫卡片頭

---

### 詩二：填報表

**七言絕句 · 〈案牘行〉**

```
寬袖輕拂鍵盤聲，
試算表中歲月深。
毛筆端坐從未動，
社畜古今一般情。
```

**白話意譯：** 寬袖古裝輕拂，鍵盤聲聲作響，在試算表裡蹉跎歲月。那支毛筆從未沾過墨水——打工人的苦，古今皆同。

**建議用途：** 第二段字幕（第1-3秒填報表場景），旁白吟誦

---

### 詩三：開會

**仿《念奴嬌》· 〈折扇詞〉**

```
大江東去，浪淘盡，
千古風流人物。
今朝格間，折扇舉，
PPT 第三頁。
聽者頻頻點頭，
手中滑鼠紋絲未動，
神思早出格。
一聲「有問題嗎？」
滿堂沉默如謁。
```

**白話意譯：** 古今多少豪傑，到了會議室都一樣——主講者滔滔不絕，聽眾頻點頭卻早已神遊，被點名時全場沉默如石。

**建議用途：** 第二段旁白（3-7秒開會場景），可分兩句顯示

### 詩四：螢幕發呆

**七言絕句 · 〈電子銅鏡〉**

```
黑屏如鏡映桃花，
山色窗光入畫斜。
人在格間心在野，
一春夢碎 deadline 下。
```

**白話意譯：** 熄滅的螢幕如古銅鏡，映出窗外桃花山色。人坐在格間，心早已飄向山野——一個美夢，碎在截止日期之下。

**建議用途：** 第二段字幕（7-10秒發呆場景），情感過渡

---

### 詩五：深夜加班

**五言律詩 · 〈夜值〉**

```
月掛西樓角，
螢光照案前。
左鄰鍵盤睡，
右友糕點牽。
燭火三更短，
報表一頁添。
問君何時畢，
天明再說吧。
```

**白話意譯：** 月亮已掛西邊，螢幕光照著桌面。左邊同事趴著睡著，右邊同事偷偷吃糕點。燭火快燒完，報表才加一頁。問你幾時才能做完？天亮了再說吧。

**建議用途：** 第三段旁白（1-4秒加班群像），配古琴音緩緩誦讀

---

### 詩六：出畫感嘆

**仿《青玉案》· 〈打工人〉**

```
東風夜放桃花落，
更吹動、宣紙角。
畫中人不知今是何年，
長街短巷，
格間燈火，
歲歲依舊過。

眾裡尋他千百度，
驀然回首，
那人還在——
螢幕前坐。
```

**白話意譯：** 春風吹落桃花，吹動畫卷一角。畫中人不知今是哪年，長街短巷，辦公室燈火，年年如此。尋遍人間，驀然回頭，那個打工人，還是坐在螢幕前。

**建議用途：** 第三段收尾字幕（7-10秒出畫場景），分兩段淡入顯示，配古琴尾音
