# 鐵鷹無人機 — Gemini 生圖提示詞

```
外形設計成老鷹形狀的軍用戰鬥無人機。機身鐵灰色金屬，機翼造型模仿鷹翼展開。機頭正面有兩個發著紅光的感應器眼睛，形似猛禽眼神兇狠。機翼下方各掛載一枚導彈。整體輪廓融合猛禽與現代無人機，流線兇悍。懸停於漆黑夜空中，紅色眼睛發出幽幽紅光照亮機身底部。邵氏1970年代香港電影美學結合復古科幻風，膠片顆粒質感，紅光從下方打光，深黑背景，高對比，略帶褪色的復古色調。
```
