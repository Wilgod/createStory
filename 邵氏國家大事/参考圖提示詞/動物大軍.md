# 動物大軍 — Gemini 生圖提示詞

```
史詩級廣角遠景：大批擬人化老鷹與獅子組成的聯合大軍，在廣闊沙漠中浩浩蕩蕩前進。畫面上半部，數以百計的擬人化大鷹展翅排成緊密軍陣，遮天蔽日飛行。畫面下半部，一排排擬人化獅子士兵手持長矛盾牌整齊行軍，揚起漫天黃沙塵柱。大軍延伸至天邊地平線。烈日沙漠，金黃沙塵瀰漫，氣勢磅礴。邵氏1970年代香港武俠電影風格，膠片顆粒質感，史詩電影寬銀幕構圖，暖金色調，高對比，略帶褪色的復古色調。
```
