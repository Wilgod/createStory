# 油桶 — Gemini 生圖提示詞

```
一個大型工業鋼鐵油桶，直立放置於沙地上。桶身黑色金屬，表面略帶光澤，有鉚釘縫線紋理。桶身正面貼著一張鮮紅色長方形標籤，標籤上印著一個超大黑色漢字「油」，字體粗黑飽滿，佔滿整張標籤，採用老式木版印刷印章風格，略帶粗糙感。四分之三側角特寫，突出「油」字標籤。邵氏1970年代香港電影風格，膠片顆粒質感，強烈單側光照明，高對比，略帶褪色的復古色調，簡單深色或沙地背景。
```
