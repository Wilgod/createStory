# 麻雀桌 — Gemini 生圖提示詞

```
溫馨家居室內夜晚場景，無人物純場景。畫面中央一張圓形木製麻將桌，四張椅子圍坐，桌面整齊排列著麻將牌。桌上方懸掛一盞暖黃色吊燈，光暈集中在桌面，四周漸暗。背景為昏暗的木質家具、小茶几配茶杯、簡單窗簾窗口。家庭生活氣息，溫暖而凌亂。邵氏1970年代香港家庭電影美學，膠片顆粒質感，暖琥珀色與黃色調，柔和陰影層次，略帶褪色的復古色調。
```
