# 啤牌局 — Gemini 生圖提示詞

```
昏暗地下室夜晚室內場景，秘密牌局氛圍，無人物純場景。畫面中央一張長方形木桌，桌面散落啤牌（中式紙牌）、幾疊籌碼，以及數個倒側的空啤酒瓶。桌上方一個裸露燈泡吊燈，發出刺眼向下的硬光，形成濃重陰影。牆壁為裸露混凝土或舊磚牆，略帶污跡。周圍擺著幾張軍用木椅。氣氛緊張壓抑。邵氏1970年代香港黑幫電影美學，膠片顆粒質感，高對比單光源，冷灰綠色調，深暗氛圍，略帶褪色的復古色調。
```
