# 橡膠果實 — 故事分鏡提示詞

**故事簡介**：女主角擁有橡膠果實能力，前往農夫家追問冬天救狐狸一事，農夫惱怒拿出RPG射擊，女主角用橡膠身體彈走火箭，再以巨拳將農夫打飛上天。

**總時長**：約 60 秒 ｜ **風格**：邵氏風格

---

```
【时长】6秒

3-6秒：切到室内视角，门大开，女主角逆光站在门框中，面带微笑望向屋内

【声音】木门推开音效，轻快背景音乐
```

## 第4段｜10秒｜女主角吹脹身體彈走火箭

```
【风格】邵氏风格，1970年代香港武侠电影质感，胶片颗粒感，夸张动作特效，鲜艳色彩，16:9
【时长】10秒

0-2秒：女主角正面广角，看到火箭從正面飞来，嘴角上扬，深吸一口气
2-5秒：慢动作特写，身体如气球急速膨胀，皮肤呈橡胶质感，全身圆滚滚
5-6秒：火箭砸中膨胀的身体，被强力弹开，镜头跟随火箭飞离
6-9秒：纯远景镜头，画面中只有火箭拖着白烟高速飞向远处田野，轰然爆炸，浓烟四起，碎石尘土漫天
9-10秒：烟尘缓缓散开，远处一片狼藉，无女主角入镜

【声音】橡皮拉伸弹性音效，轰隆爆炸声，夸张弹跳音，欢快BGM恢复
```

```
【风格】邵氏风格，1970年代香港武侠电影质感，胶片颗粒感，喜剧夸张收尾，蓝天白云，16:9
```

## 第7段｜13秒｜神龍大戰魔動王燒鴨王

```
【风格】邵氏风格，1970年代香港武侠电影质感，胶片颗粒感，史诗对决，农夫家后园，非雪地，16:9
【时长】13秒
【地点】农夫家后园：泥土地面，枯草围栏，木柴堆，傍晚橙红天色

0-2秒：广角全景，镜头缓缓推入农夫家后园，平静的泥土庭院，忽然地面震动，裂缝从中央延伸
2-4秒：一条巨龙从地下破土而出，鳞片金光闪烁，龙首仰天长啸，震得木栅栏碎裂
4-6秒：对面出现魔动王燒鴨王，身披烧鸭色泽的暗红铠甲，手持油亮大刀，踏步而来，脚步踩碎石板
6-8秒：双方对峙特写交替切镜——神龙眼睛特写（金瞳）→ 燒鴨王眼睛特写（赤瞳）→ 双人远景对峙
8-11秒：神龙喷出火焰冲向燒鴨王，燒鴨王举刀迎斩，火焰被劈成两道，爆炸气浪席卷整个后园，木柴堆炸飞
11-13秒：烟尘中两道巨影缠斗，镜头拉远，女主角站在后园入口望着这一切，耸耸肩转身离开

【声音】地面震裂音，龙啸声，魔王重步踏地声，刀劈火焰爆裂声，混战轰鸣声，BGM转为史诗对决鼓点
```
