# 英超領隊WhatsApp群組 — 故事分鏡提示詞

**故事簡介**：英超各大球隊領隊組成一個WhatsApp群組。Arsenal領隊Mikel Arteta豪氣宣佈追求四冠王，卻一冠一冠地丟失。Pep Guardiola、Arne Slot、Ruben Amorim、Ange Postecoglou等勁敵以「恭喜」為名，字字補刀，眼睜睜見證Arteta由四冠王→三冠王→二冠王的史詩式「成就達成」。

**角色名單**：

| 角色 | 球隊 | 頭像Emoji | 性格定位 |
|------|------|-----------|----------|
| Mikel Arteta | Arsenal 🔴⚪ | 🏖️ | 主角，愈來愈沉默，最後一句驚天動地 |
| Pep Guardiola | Man City 🔵⚪ | 🧠 | 哲學家，長篇大論「鼓勵」，實為補刀王 |
| Arne Slot | Liverpool 🔴 | 🤌 | 冷靜荷蘭人，短短一句勝千言 |
| Ruben Amorim | Man United 🔴⚫ | 😔 | 自身難保，卻仍忍不住插嘴 |
| Thomas Frank | Tottenham ⚪ | 🦅 | 澳洲佬，激進豪邁，永遠搶先發言 |

**總時長**：約 60 秒｜**風格**：手機WhatsApp聊天室動態畫面 × 寫實UI × 喜劇

## 第1段｜10秒｜賽季開幕 — Arteta宣佈四冠王目標

```
【風格】手機直立畫面9:16，WhatsApp群組介面，白色對話泡，Android/iOS混合風，清晰字體，群組名稱「🏴󠁧󠁢󠁥󠁮󠁧󠁿 英超領隊群組」置頂顯示，淺灰背景，自然室內光感
【時長】10秒

0-2秒：鏡頭緩慢拉近一部手機螢幕，WhatsApp群組對話欄空白，群組圖示顯示五位領隊的足球卡通頭像。右上角顯示「5位成員」。
2-5秒：Arteta（🏖️ Mikel）發送訊息，訊息泡逐字打出——
「今個賽季，我哋Arsenal追求四冠王。EPL、聯賽盃、足總盃、歐聯。一個都唔少 💪🔴」
訊息發送後出現雙藍剔。
5-8秒：Pep（🧠 Pep）即時回覆——
「Mikel，我幫你數好咗，係四個。非常正確 🧠」
緊接Ange（🦅 Ange）——
「哇，比我更勇！我只係追一個 🦅」
8-10秒：群組顯示「Mikel 正在輸入...」三點閃爍，然後消失。Arteta沒有回覆。

0-2秒：群組最頂部出現一條系統式貼文（灰底白字居中）——
「⚽ Arsenal 0 - 1 Newcastle｜聯賽盃｜Arteta沉默中」
2-5秒：Pep發出長訊息，文字逐行滾出——
「Mikel，恭喜你！三冠王目標達成第一步。
減少目標，係成功嘅開始。
我好欣賞你嘅適應力 👏🧠」
5-7秒：Arne（🤌 Arne）回覆一句——
「三個都係三個 🤌」
7-9秒：Ruben（😔 Ruben）打出——
「加油Mikel。我……懂這種感覺 😔」
旁邊Ange補一句：「哈哈 🦅」
9-10秒：群組顯示「Mikel 正在輸入...」持續閃爍，畫面定格。


0-3秒：「Mikel 正在輸入...」持續閃爍，持續時間誇張地長。鏡頭稍微推近文字區。
3-5秒：Arteta終於發出訊息——
「我哋仍然專注。三個目標。我哋係Arsenal 💪」
發送後出現單灰剔（未讀）。
5-7秒：Ange即時搶先回覆——
「對！專注喺三個！哦等等，係咪仲有三個先？🦅」
緊接群組內爆出一排Emoji反應：😂😂😂👀
7-8秒：Pep慢慢加上一個👍 Emoji反應在Arteta訊息上。畫面稍微震動效果。

【聲音】打字聲（長而緩慢），「叮」聲，笑聲Emoji音效（卡通化），群組震動音
【參考】無
```

## 第4段｜10秒｜歐聯/足總盃出局 — 三冠王變二冠王

```
【風格】同前，手機WhatsApp，9:16，訊息快速滾動，螢幕輕微抖動感
【時長】10秒

0-2秒：另一條系統式貼文彈出（紅底白字，更顯眼）——
「🚨 Arsenal 歐聯出局｜三冠王正式成為歷史」
2-5秒：Pep發出訊息，加粗感覺（多個感嘆號）——
「Mikel！！恭喜你！！！
二冠王目標達成！！
你係一個有計劃嘅人 🧠🧠🧠」
5-7秒：Arne冷靜發出——
「二冠王都係冠王。基本上 🤌」
7-9秒：Ruben打出——
「二個比零個多。😔」
訊息旁出現細細灰字標注：*（Man United今季係零）*
9-10秒：鏡頭拉遠一點，可見整個手機畫面，群組通知數字不斷跳升。

【聲音】訊息「叮叮叮叮」密集聲，背景新聞播報音效片段（極輕：「Arsenal已被淘汰...」），Ruben訊息後短暫靜止
【參考】無
```

「我早就話過㗎！哈哈！🦅」
「我早就話過㗎！哈哈！🦅」
「我早就話過㗎！哈哈！🦅」
「我早就話過㗎！哈哈！🦅」
3-5秒：Pep發送一張「截圖」風格訊息（灰底長文預覽框）——

二冠王係一個哲學上嘅勝利。
你由四個目標進化成兩個目標，

【聲音】連環「叮叮叮叮叮」訊息聲（密集），群組反應Emoji音效，「此訊息已刪除」出現時有短暫靜音
【參考】無

## 第6段｜10秒｜結局 — Arteta最後一擊

```
【風格】同前，9:16，節奏由快轉慢，最後一句訊息特別放大顯示，震動效果
【時長】10秒

0-3秒：群組靜止，無新訊息。「Arteta 正在輸入...」出現又消失，消失又出現，循環兩次。所有人已讀標記亮起，靜待。
3-5秒：Arteta發出訊息——訊息泡比平時稍大，字體清晰——
「下個賽季，我哋追五冠王 💪🔴」
發送音效。
5-7秒：群組靜止三秒，無任何回覆。畫面定格。
7-9秒：突然爆出五條訊息同時湧入——
Pep：「😱」
Arne：「😱」
Ruben：「😱」
Ange：「😱😱😱😱😱」
Pep補發：「我愛你Mikel 🧠❤️」
9-10秒：畫面緩緩淡出，手機螢幕亮度降低，WhatsApp通知聲「叮」一聲，黑屏。字幕淡入：
「下個賽季，見 🏆」

【聲音】打字聲（緩慢沉穩），訊息發送音，三秒靜默，然後「叮叮叮叮叮」爆炸式回覆聲，最後一聲「叮」，淡出
【參考】無
```

## 製作備注

**視覺串接方式**
- 全片以手機WhatsApp介面為主視角，鏡頭在手機螢幕上推拉
- 每段承接上一段的對話記錄，訊息由下往上堆疊
- 第2、4段系統式通知可設計為全寬灰/紅色橫幅，插入對話流中間

**串接順序**
```
第1段：純UI動畫生成 → 視頻1
第2段：@視頻1 延長 → 視頻2
第3段：@視頻2 延長 → 視頻3
第4段：@視頻3 延長 → 視頻4
第5段：@視頻4 延長 → 視頻5
第6段：@視頻5 延長 → 視頻6（完結）
```
